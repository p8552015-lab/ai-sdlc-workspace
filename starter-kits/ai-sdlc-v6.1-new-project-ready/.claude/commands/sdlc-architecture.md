# /sdlc-architecture

Use before bootstrapping a new project or when changing architecture.

Claude Code must:

1. Compare 2-3 architecture options.
2. Evaluate speed, maintainability, deployment complexity, security, cost, and scalability.
3. Recommend one option.
4. Create/update `docs/architecture.md` and ADRs under `docs/adrs/`.
5. Request human approval before bootstrap or implementation.
